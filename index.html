Illustrative Risk Engine Snippet

(Python – Probability of Default model)


import numpy as np

def probability_of_default(score: float) -> float:
    """
    Simple logistic PD model.
    score: credit score (higher is better)
    """
    # Intercept and coefficient are illustrative
    intercept = -5.2
    beta_score = -0.015

    logit = intercept + beta_score * score
    pd = 1 / (1 + np.exp(-logit))
    return float(pd)

# Example: client with score 720
print(probability_of_default(720))

Core Financial Formulas

Net Present Value (NPV) of a cash-flow stream:

\[ \text{NPV} = \sum_{t=0}^{T} \frac{C_t}{(1 + r)^t} \]

Return on Equity (ROE):

\[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

Cost-to-Income Ratio:

\[ \text{C/I} = \frac{\text{Operating Expenses}}{\text{Operating Income}} \]
